- To use annontations `@EnableJms` and `@JmsListener`
  ```xml
    <dependency>
        <groupId>org.springframework</groupId>
        <artifactId>spring-jms</artifactId>
        <version>6.2.0</version>
    </dependency>
  ```
- To use `TextMessage` class
  ```xml
      <dependency>
          <groupId>org.springframework</groupId>
          <artifactId>spring-jms</artifactId>
          <version>6.2.0</version>
      </dependency>
  ```
